# CSE351 Lab Workspace

This repository contains coursework for CSE351, organized by lab. Each folder holds the source code and writeups for one assignment:

- `lab 1a/` - pointer and memory exercises in C
- `lab 1b/` - a store/aisle management lab with provided tests and a Makefile
- `lab 4/` - cache-friendly matrix transpose work
- `lab 5/` - dynamic memory allocator implementation
