5) Difficulty + why (past experience): The most difficult part for me was tracking which registers held which values and provided which functions at any given point in the code. Unlike variables in languages like C and Java that we learned before, which have clear names and uses, registers in assembly get reused constantly for different purposes (sometimes holding a pointer, sometimes a loop counter, sometimes a return value). This was difficult because for example in phase 5, I had to carefully track that rbx held the input string pointer, rax was the loop index, and rdx was being reused first as the current character then as the table lookup result.
